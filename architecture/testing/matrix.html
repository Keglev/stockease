Matrix Overview

What is a Coverage Matrix?

A coverage matrix crosses two dimensions to show what combinations are tested:

           Unit    Slice   Integration
Layer      Test    Test    Test
─────────────────────────────────────
Controller  ✅      ✅       ⏳
Service     ❌      ❌       ❌
Repository  ❌      ⏳       ❌
Entity      ❌      ⏳       ⏳
Security    ✅      ✅       ⏳
─────────────────────────────────────

✅ = Covered    ⏳ = Partial    ❌ = Not Covered
